# TravelPlanner 设计上下文

## 项目概述
Multi-Agent 智能旅行规划系统 - 基于 LangGraph 的 AI 旅行规划工具，通过结构化表单输入用户需求，并行采集数据，协调冲突输出最佳行程方案。

## 用户画像
- **主要用户**：25-40 岁的城市白领、旅行爱好者
- **使用场景**：计划周末短途游、节假日出行、出差间隙游玩
- **核心需求**：快速获得个性化、合理的旅行方案
- **情感期望**：轻松愉悦 - 让规划旅行本身成为一种享受

## 品牌个性
**轻松 · 智能 · 精致**

就像有一个懂旅行的朋友，帮你把一切安排得妥妥当当，你只需要享受旅程。

## 交互原则

### 1. 即时反馈
- 按钮悬停：scale(1.02) + shadow 增强
- 点击：scale(0.98)
- 表单聚焦：蓝色光环动画
- 加载状态：骨架屏 > 内容淡入

### 2. 渐进呈现
- 页面进入：元素依次淡入上滑
- 内容加载：骨架屏占位
- 数据更新：数字滚动动画

### 3. 流畅转场
- 页面切换：共享元素过渡
- Tab 切换：滑动动画
- 列表展开：高度自适应动画

### 4. 触觉反馈
- 成功操作：轻微缩放 + 颜色闪烁
- 错误操作：轻微抖动
- 完成状态：对勾动画

## 页面结构

### 首页 (Home)
1. **Hero 区域** - 大标题 + 价值主张 + 主 CTA
2. **特色展示** - 3 个核心卖点卡片
3. **规划表单** - TripComposerForm（核心交互）
4. **信任背书** - 用户评价/数据展示

### 规划页 (Planning)
1. **进度头部** - 目的地 + 行程概要
2. **可视化进度** - 大进度条 + 百分比
3. **Agent 状态** - 各 Agent 执行状态卡片
4. **实时日志** - 流式输出区域
5. **结果预览** - 景点/酒店数量统计

### 结果页 (Result)
1. **行程概览** - 城市 + 天数 + 标签
2. **天气栏** - 每日天气预报
3. **时间轴** - 天数切换 Tab
4. **地图视图** - 高德地图 + 路线标记
5. **日程详情** - 每日景点/餐饮/住宿卡片
6. **操作按钮** - 保存/分享/重新规划
